# 主题定制

NavLink 提供丰富的主题定制选项，打造专属于您的导航站。

## 主题配色

### 主色调

设置按钮、链接等强调色：

- 预设多种配色方案
- 支持自定义颜色
- 一键应用全站

### 文字颜色

- 主要文字颜色
- 次要文字颜色
- 链接文字颜色

### 字体大小

全局字体大小调节：

- 小号：适合紧凑布局
- 中号：默认推荐
- 大号：适合大屏展示

## 背景设置

### 纯色背景

- 单色背景
- 渐变背景
- 自定义渐变方向

### 图片背景

- 上传本地图片
- 填写网络图片 URL
- 支持模糊效果
- 支持透明度调节

## 导航栏样式

三种导航栏风格可选：

### 沉浸式

- 透明背景
- 融入页面背景
- 适合图片背景

### 毛玻璃

- 半透明模糊效果
- 现代感设计
- 适合各种背景

### 独立背景

- 纯色导航栏
- 与页面背景分离
- 适合纯色背景

## 首页布局

### 搜索区域 (Hero)

- 自定义格言标题
- 自定义副标题
- 调整高度

### 分类展示

- 网格布局
- 列表布局
- 卡片大小调节

## 侧边栏

### 个人名片

- 头像设置
- 昵称/标题
- 简介文案
- 背景颜色

### 社交链接

- 添加社交媒体图标
- 自定义链接
- 图标颜色

### 热点榜单

- 显示实时热点
- 自定义数据源
- 调整显示数量

## 其他设置

### Favicon

浏览器标签页图标

### Logo

导航栏显示的 Logo 图片

### 页脚

自定义版权信息和备案号

## 配置导出

所有主题配置可以导出为 JSON 文件，方便备份和迁移。
