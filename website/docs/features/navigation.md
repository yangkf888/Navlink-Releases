# 智能导航

NavLink 提供强大的导航管理功能，让您轻松组织和访问常用链接。

## 多级分类

支持灵活的分类层级结构：

### 一级分类

- 左侧导航栏展示
- 点击快速定位
- 支持图标设置
- 拖拽调整顺序

### 子分类 Tab

部分分类支持多个子 Tab：

- 适合同类别的不同分组
- Tab 切换展示不同链接
- 例如：「开发工具」→「前端」「后端」「数据库」

## 链接管理

### 添加链接

1. 点击「添加链接」按钮
2. 输入 URL
3. 系统自动识别：
   - 网站标题
   - 网站描述
   - 网站图标 (Favicon)
4. 保存即可

### 链接信息

每个链接包含：

| 字段 | 说明 |
|------|------|
| 标题 | 链接显示名称 |
| URL | 目标地址 |
| 描述 | 鼠标悬浮提示 |
| 图标 | FontAwesome 图标或图片 URL |
| 颜色 | 图标背景色 |

### 拖拽排序

- 长按链接卡片可拖拽
- 支持跨分类移动
- 实时保存顺序

## 链接健康检测

NavLink 会自动检测链接的可用性：

### 自动检测

- 定时扫描所有链接
- 检测 HTTP 状态码
- 标记失效链接（红色警告）

### 手动检测

- 后台可一键全站检测
- 查看检测结果报告
- 批量清理失效链接

## 访问控制

### 分类隐藏

可设置分类为「未登录隐藏」：

- 该分类下的链接仅管理员可见
- 适合存放私密链接
- 访客看不到这些内容

### 链接权限

- 公开链接：所有人可见
- 私密链接：登录后可见
