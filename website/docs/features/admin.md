# 后台管理

NavLink 提供功能完善的后台管理面板，让您轻松管理导航站的各项配置。

## 进入后台

点击页面右上角的**头像图标**即可进入后台管理。

## 模块概览

后台分为以下几个主要模块：

| 模块 | 功能 |
|------|------|
| 基础设置 | 站点信息、主题配色、背景设置 |
| 首屏配置 | 搜索引擎、热词推荐、格言设置 |
| 分类管理 | 分类和链接的增删改查 |
| 侧边栏 | 个人名片、社交链接、热点榜单 |
| 系统设置 | 数据管理、链接健康、系统升级 |

## 基础设置

### 站点信息

- **Logo**: 导航栏显示的网站图标
- **Favicon**: 浏览器标签页图标
- **网站标题**: 页面标题
- **页脚版权**: 底部版权信息

### 主题配置

- 主色调选择
- 背景模式（纯色/图片）
- 导航栏样式
- 字体大小

## 分类管理

### 分类操作

- ✅ 新建分类
- ✅ 编辑分类
- ✅ 删除分类
- ✅ 拖拽排序
- ✅ 启用子 Tab

### 链接操作

- ✅ 添加链接（智能识别）
- ✅ 编辑链接
- ✅ 删除链接
- ✅ 拖拽排序
- ✅ 跨分类移动

## 数据管理

### 导出配置

一键导出完整配置为 JSON 文件：

- 站点设置
- 分类和链接
- 主题配置
- 侧边栏设置

### 导入配置

支持两种导入方式：

1. 上传 JSON 文件
2. 直接粘贴 JSON 内容

::: warning ⚠️ 注意
导入会覆盖当前配置，请先备份！
:::

### 本地备份

浏览器自动保存最近 5 个快照到 localStorage，可随时恢复。

## 链接健康

### 全站检测

一键扫描所有链接的可用性：

- 检测 HTTP 响应状态
- 标记失效链接
- 生成检测报告

### 自动巡检

开启后系统每天自动检测，并标记失效链接。

### 批量清理

检测完成后可一键删除所有失效链接。

## 系统升级

### 在线更新

（仅 Docker 环境）

1. 检查最新版本
2. 一键拉取新镜像
3. 自动重启容器
4. 升级前自动备份

## 用户管理

### 多用户支持

- 管理员账号
- 普通用户账号
- 权限分级控制

### 密码修改

后台可随时修改登录密码。

## AI 设置

### 提供商配置

添加和管理 AI 服务提供商：

- Base URL
- API Key
- 模型选择

### 知识库

启用/禁用知识库增强功能。
