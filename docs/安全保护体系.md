## 🚀 快速开始

### 开发环境
```bash
# 使用原始代码（便于调试）
npm run dev:all
```

### 生产构建
```bash
# 自动混淆后端代码
npm run build:all

# 构建 Docker 镜像（已集成混淆）
npm run build:docker
```

## ⚠️ 重要提示

1. **不要提交混淆后的代码到 Git**
   - `dist-server/` 已在 `.gitignore` 中

2. **保护私钥安全**
   - 不要将 `scripts/sign-license.js` 中的私钥提交到仓库

3. **定期更新**
   - 定期检查混淆工具更新
   - 关注安全漏洞公告
